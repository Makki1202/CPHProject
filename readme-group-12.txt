SuperStringWithExpansion
=========================================

Group: 12
Members: [s215525, s242908, s243593]

-----------------------------------------
Requirements
-----------------------------------------
- C++ compiler with at least C++17 support (e.g., g++ 7 or higher)
- Standard input/output only

-----------------------------------------
Compilation (Linux / macOS)
-----------------------------------------

g++ -std=c++17 -o SuperStringWithExpansion Project.cpp

-----------------------------------------
Compilation (Windows)
-----------------------------------------

g++ -std=c++17 -o SuperStringWithExpansion.exe Project.cpp

-----------------------------------------
Execution (Linux / macOS)
-----------------------------------------

./SuperStringWithExpansion < tests/test01.swe

-----------------------------------------
Execution (Windows)
-----------------------------------------

SuperStringWithExpansion.exe < tests/test01.swe
